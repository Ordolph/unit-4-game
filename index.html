Welcome to Crystal Collector!

How to play:

Push the start button in order to start the game!

The objective of this game is to match your score to "Your Number" without going over. This can be achieved by clicking on the gems. At the beginning of each game, each gem is assigned a random number 1-12, clicking the gem will add it's value to your score. Have fun!
